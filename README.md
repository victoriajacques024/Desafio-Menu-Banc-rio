Este projeto é um pequeno sistema bancário em Java desenvolvido como exercício de lógica de programação. 
Ele simula as principais operações de uma conta corrente, permitindo que o usuário interaja pelo terminal.

O programa utiliza a classe Scanner para ler as entradas do usuário no terminal.
Com base na opção escolhida, ele executa diferentes ações, atualizando o saldo conforme as operações de transferência e depósito.
